# Evidencias de Perplexity AI

Este directorio contiene las evidencias relacionadas con el uso de Perplexity AI en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:

- Búsqueda técnica utilizando Perplexity AI con citaciones verificables.
- Visualización de las fuentes recuperadas (arxiv, cisco, NCBI, GitHub, entre otras).
- Análisis de confiabilidad de las fuentes citadas en tiempo real.
- Comparación entre el enfoque de Perplexity y los motores de búsqueda tradicionales.
- Identificación de cómo Perplexity combina LLM + búsqueda para mejorar respuestas basadas en evidencia.
