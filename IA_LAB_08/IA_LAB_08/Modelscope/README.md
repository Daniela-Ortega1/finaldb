# Evidencias de Modelscope

Este directorio contiene las evidencias relacionadas con el uso de **Modelscope** en el Laboratorio 8 – Inteligencia Artificial.

## Evidencias incluidas

- Exploración del catálogo de modelos disponibles en Modelscope.
- Ejecución de un modelo de visión, audio o NLP desde la plataforma web.
- Pruebas de inferencia con modelos preentrenados usando la interfaz online.
- Análisis del rendimiento de un modelo básico recomendado por la plataforma.
- Ajuste simple de parámetros según el tipo de tarea (clasificación, generación, segmentación, etc.).
- Comparación entre diferentes modelos sugeridos por Modelscope.
- Exportación de resultados o capturas generadas durante las pruebas como evidencia.
