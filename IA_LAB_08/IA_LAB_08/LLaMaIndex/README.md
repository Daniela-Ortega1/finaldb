# Evidencias de LLaMaIndex

Este directorio contiene las evidencias relacionadas con el uso de **LLaMaIndex** en el Laboratorio 8 – Inteligencia Artificial.

## Evidencias incluidas

- Construcción de un índice básico a partir de documentos cargados manualmente.  
- Uso de un LLM para consultar información del índice.  
- Ejecución de consultas simples y avanzadas sobre el documento cargado.  
- Comparación entre respuestas directas del modelo y respuestas basadas en el índice.  
- Prueba del módulo de “Query Engine” para obtener respuestas contextualizadas.  
- Exportación del log de consultas o pantallazos como evidencia del uso.
