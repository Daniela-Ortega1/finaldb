# Evidencias de Papers With Code

Este directorio contiene las evidencias relacionadas con el uso de Papers With Code en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:

- Consulta inicial del catálogo de artículos asociados a la tarea MNIST.
- Selección de un paper con implementación reproducible disponible.
- Revisión de la página oficial del paper en HuggingFace Papers (Papers With Code).
- Acceso al repositorio oficial del experimento en GitHub.
- Carga del dataset Fashion-MNIST dentro de un Kaggle Notebook.
- Ejecución parcial del experimento siguiendo las instrucciones del repositorio.
- Entrenamiento de un modelo CNN utilizando los datos proporcionados.
- Comparación básica entre los resultados obtenidos y los reportados por los autores.
