# Evidencias de Devin AI

Este directorio contiene las evidencias relacionadas con el uso de **Devin AI** en el Laboratorio 8 – Inteligencia Artificial.

## Evidencias incluidas

- Exploración de la interfaz de Devin AI como “AI Software Engineer”.  
- Creación de una tarea simple para que Devin la ejecute automáticamente.  
- Observación de cómo la IA escribe código, crea archivos y ejecuta pruebas.  
- Análisis del flujo de trabajo autónomo generado por Devin.  
- Comparación entre el enfoque de Devin y los copilotos tradicionales.  
- Exportación de capturas mostrando el proceso automático realizado por la herramienta.
