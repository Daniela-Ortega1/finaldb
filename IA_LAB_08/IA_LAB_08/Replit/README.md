# Evidencias de Replit

Este directorio contiene las evidencias relacionadas con el uso de **Replit** en el Laboratorio 8 – Inteligencia Artificial.

## Evidencias incluidas

- Creación de un nuevo proyecto en Replit desde cero.  
- Ejecución de un script en Python dentro del entorno online.  
- Uso del asistente de IA integrado para mejorar o completar código.  
- Instalación de dependencias usando el panel de paquetes.  
- Prueba del sistema de auto-ejecución y consola integrada.  
- Exportación de salidas o capturas de pantalla del proyecto ejecutado.
