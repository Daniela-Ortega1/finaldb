# Evidencias de GitHub Copilot

Este directorio contiene las evidencias relacionadas con el uso de **GitHub Copilot** en el Laboratorio 8 – Inteligencia Artificial.

## Evidencias incluidas

- Generación automatizada de código en Python a partir de comentarios.  
- Sugerencias de completado inteligente dentro del editor de código.  
- Creación de funciones basadas en descripciones en lenguaje natural.  
- Corrección de errores y optimización del código sugerida por Copilot.  
- Ejecución de un pequeño script generado con ayuda del modelo.  
- Comparación entre escribir código manualmente y con asistencia de Copilot.
