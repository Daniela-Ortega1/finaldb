# Evidencias de Hugging Face

Este directorio contiene las evidencias relacionadas con el uso de Hugging Face en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:
- Exploración del catálogo de modelos.
- Selección de modelos de NLP, visión y audio.
- Ejecución de un modelo en Google Colab.
- Creación y despliegue de un Hugging Face Space.
- Pruebas de inferencia con modelos como FLAN-T5-Small.
