# Evidencias de Google AI Studio

Este directorio contiene las evidencias relacionadas con el uso de Google AI Studio en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:

- Pruebas en el Playground del modelo Gemini.
- Ejecución de prompts técnicos y creativos.
- Comparación del efecto de la temperatura (0.2 vs 1.0).
- Ajustes de parámetros como top-p y temperatura.
- Observación del comportamiento probabilístico del modelo según el tipo de prompt.
