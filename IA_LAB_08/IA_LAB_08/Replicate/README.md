Evidencias de Replicate

Este directorio contiene las evidencias relacionadas con el uso de Replicate en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:

Exploración del catálogo de modelos disponibles en Replicate.

Ejecución de un modelo de generación de imágenes mediante API o interfaz web.

Ajuste de parámetros como prompt, negative prompt, steps y cfg_scale.

Comparación entre diferentes modelos de generación (Stable Diffusion, Flux, u otros).

Generación de imágenes basadas en instrucciones del proyecto MORITA.

Exportación de las imágenes generadas como evidencia.
