# Caso Integrador – MORITA

Este directorio contiene las evidencias correspondientes al **Caso Integrador** del Laboratorio 8 – Inteligencia Artificial.

## Contenido incluido

- Implementación de ejemplos prácticos relacionados con el proyecto **MORITA**.  
- Uso de modelos de IA para generar:  
  - Imágenes relacionadas con el proceso productivo.  
  - Descripciones de módulos.  
  - Código o pruebas aplicables al sistema.  
- Comparación entre las diferentes herramientas utilizadas durante el laboratorio.  
- Reflexión final sobre el impacto de la IA en el desarrollo del proyecto MORITA.  
- Capturas de las evidencias generadas durante la ejecución del caso integrador.
