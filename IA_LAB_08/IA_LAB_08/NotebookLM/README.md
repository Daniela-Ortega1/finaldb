# Evidencias de NotebookLM

Este directorio contiene las evidencias relacionadas con el uso de NotebookLM en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:
- Carga del documento del proyecto MORITA en NotebookLM.
- Análisis automatizado del contenido.
- Consulta sobre objetivo general y objetivos específicos usando solo la fuente cargada.
