# Evidencias de Kaggle

Este directorio contiene las evidencias relacionadas con el uso de Kaggle en el Laboratorio 8 – Inteligencia Artificial.

Evidencias incluidas:

- Selección del dataset *Iris Dataset* dentro de Kaggle.
- Creación de un notebook para realizar el análisis exploratorio de datos (EDA).
- Carga del archivo `iris.csv` dentro del notebook.
- Visualización inicial de los datos con `df.head()`.
- Inspección de tipos de datos y valores nulos usando `df.info()`.
- Estadísticos descriptivos generados con `df.describe()`.
- Gráfico *pairplot* con Seaborn para explorar relaciones entre variables.
- Mapa de calor de correlación para analizar relaciones lineales.
- Publicación del notebook ejecutado en el perfil de Kaggle.
